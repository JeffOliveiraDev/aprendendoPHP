Gerar a imagem a partir do dockerfile
<br>

```
docker build -t curso-php .
```

Rodar a imagem gerada

```
docker run -it --rm --name curso-php-run curso-php
```

Testando na web(entrar no localhost selecionado)
http://localhost:80
